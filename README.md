# 📝 Markdown 微信公众号编辑器

一个专为微信公众号设计的在线 Markdown 编辑器，支持实时预览和一键复制富文本格式。

## ✨ 功能特点

- 📝 **实时编辑预览** - 左侧编辑，右侧实时预览
- 🎨 **微信样式优化** - 完美适配微信公众号样式
- 📋 **一键复制富文本** - 直接粘贴到微信公众号后台
- 💻 **代码高亮** - 支持多种编程语言语法高亮
- 🌙 **深色模式** - 保护眼睛，支持明暗主题切换
- 💾 **本地存储** - 自动保存编辑内容，防止丢失
- 📥 **导出功能** - 支持下载 Markdown 文件
- 📱 **响应式设计** - 完美支持移动端和桌面端
- ⌨️ **快捷键支持** - Ctrl/Cmd + S 快速保存

## 🚀 快速开始

### 在线使用

直接访问：[https://dongboshi.github.io/md_to_weichat/](https://dongboshi.github.io/md_to_weichat/)

### 本地部署

1. 克隆仓库
```bash
git clone https://github.com/DongboShi/md_to_weichat.git
cd md_to_weichat
```

2. 使用浏览器打开 `index.html` 即可

> 注意：本项目为纯静态页面，无需安装依赖或构建步骤。

## 📖 使用方法

1. **编辑 Markdown**
   - 在左侧编辑器输入 Markdown 内容
   - 支持 Tab 键缩进（插入 4 个空格）

2. **查看预览**
   - 右侧实时显示渲染后的效果
   - 样式已针对微信公众号优化

3. **复制到微信**
   - 点击「📋 复制富文本」按钮
   - 打开微信公众号后台编辑器
   - 直接粘贴（Ctrl+V 或 Cmd+V）

4. **其他功能**
   - 点击「🗑️ 清空内容」清除所有文本
   - 点击「💾 下载 MD」保存为 Markdown 文件
   - 使用「🌙 深色模式」切换主题

## 🎨 支持的 Markdown 语法

- **标题** - `# H1` 到 `###### H6`
- **强调** - `**粗体**` 和 `*斜体*`
- **列表** - 有序列表和无序列表
- **链接** - `[文字](URL)`
- **图片** - `![alt](URL)`
- **代码** - 行内代码和代码块
- **引用** - `> 引用文字`
- **表格** - Markdown 表格语法
- **分割线** - `---`

## 💡 样式特点

专为微信公众号优化的样式：

- **标题样式**
  - H1：绿色底边框
  - H2：左侧绿色边框  
  
- **文本样式**
  - 粗体：微信绿色高亮
  - 斜体：微信蓝色
  - 链接：蓝色带下划线

- **代码样式**
  - 行内代码：红色背景
  - 代码块：灰色背景 + 语法高亮

- **其他元素**
  - 引用块：左侧绿色边框 + 灰色背景
  - 表格：绿色表头
  - 图片：自动居中 + 圆角阴影

## 🛠️ 技术栈

- **纯前端实现** - HTML + CSS + JavaScript
- **Marked.js** - Markdown 解析引擎
- **Highlight.js** - 代码语法高亮
- **Clipboard API** - 富文本复制功能
- **LocalStorage** - 本地内容保存

## ⚠️ 注意事项

1. **复制功能** - 需要 HTTPS 环境或 localhost 才能使用 Clipboard API
2. **图片处理** - 微信公众号不支持外链图片，需手动上传
3. **样式限制** - 微信会过滤部分 CSS 样式，复杂样式可能显示不一致
4. **本地存储** - 清除浏览器数据会丢失已保存内容，建议及时下载备份
